# КРАФТ-ДЕВАЙС

Маркетплейс-платформа, яка з'єднує виробників крафтової продукції з майстрами, що виготовляють унікальне обладнання на замовлення.

## Проблема та рішення

Виробники крафтової продукції (пиво, дистиляти, ефірні олії, кераміка тощо) потребують обладнання невеликої продуктивності, індивідуального дизайну та конструкції. Масові виробники таке не роблять. Водночас в країні достатньо кваліфікованих майстрів, які готові виконати якісне обладнання за персональними замовленнями.

**КРАФТ-ДЕВАЙС** вирішує цю проблему — замовники публікують свої потреби, а виконавці змагаються за найкращу ціну та умови.

## Основні можливості

### Для замовників
- **Тендерна система** — створюйте замовлення з детальним описом, бюджетом та дедлайном
- **Конкурентні торги** — виконавці подають ставки, ви обираєте найкращу пропозицію
- **Онлайн-конструктор** — завантажуйте креслення з AutoCAD, SolidWorks, Компас-3D та інших CAD-систем
- **Категорії** — зручний пошук по категоріях: металообробка, деревообробка, харчове обладнання, скло та кераміка, текстиль, фільтрація, пакування

### Для виконавців (майстрів)
- **Портфоліо** — демонструйте свої кращі роботи з фото, описом матеріалів та термінами виготовлення
- **Система ставок** — подавайте пропозиції на тендери з ціною, термінами та описом підходу
- **Рейтинг** — система оцінок для побудови репутації
- **Міжнародна участь** — платформа відкрита для виконавців з будь-якої країни (в т.ч. Китай)

### Інформаційні сторінки
- **Як це працює** — покрокова інструкція для нових користувачів
- **FAQ** — відповіді на найпоширеніші питання
- **Контакти** — форма зворотного зв'язку та контактна інформація

## Швидкий старт

### Локальна розробка

**Backend (порт 8000):**

```bash
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

**Frontend (порт 3000):**

```bash
cd frontend
npm install
npm run dev
```

Після запуску:
- Сайт: http://localhost:3000
- API документація (Swagger): http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

## API ендпоінти

### Авторизація

| Метод | URL | Опис |
|-------|-----|------|
| POST | `/api/auth/register` | Реєстрація нового користувача |
| POST | `/api/auth/login` | Вхід (повертає JWT токен) |
| GET | `/api/auth/me` | Дані поточного користувача (потрібен токен) |

### Тендери

| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/tenders/` | Список тендерів (фільтр по категорії, пошук) |
| POST | `/api/tenders/` | Створити тендер (авторизація) |
| GET | `/api/tenders/{id}` | Деталі тендера |
| GET | `/api/tenders/{id}/bids` | Ставки на тендер |
| POST | `/api/tenders/{id}/bids` | Подати ставку (виконавець) |
| POST | `/api/tenders/{id}/bids/{bid_id}/accept` | Прийняти ставку (замовник) |

### Портфоліо

| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/portfolio/` | Список робіт виконавців |
| POST | `/api/portfolio/` | Додати роботу (виконавець) |
| POST | `/api/portfolio/{id}/upload-image` | Завантажити фото до роботи |

### Інше

| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/categories/` | Список категорій обладнання |
| GET | `/api/stats` | Статистика платформи (кількість виконавців, тендерів тощо) |
| POST | `/api/upload` | Завантажити файл (креслення, зображення) |

## Як працює тендерна система

1. **Замовник** реєструється та створює тендер з описом, бюджетом (мін/макс), дедлайном та категорією
2. **Виконавці** переглядають доступні тендери та подають ставки зі своєю ціною, термінами та описом підходу
3. **Замовник** порівнює ставки — бачить ціну, терміни, рейтинг та портфоліо кожного виконавця
4. **Замовник обирає** найкращу ставку та приймає її
5. Виконавець отримує замовлення, тендер закривається

## Ролі користувачів

| Роль | Можливості |
|------|------------|
| **Замовник** (customer) | Створення тендерів, перегляд ставок, прийняття пропозицій, завантаження креслень |
| **Виконавець** (manufacturer) | Портфоліо, подача ставок на тендери, завантаження фото робіт |
| **Адмін** (admin) | Керування платформою |

## Категорії обладнання

- **Металообробка** — ЧПУ-верстати, зварювальне обладнання, лазерне різання
- **Деревообробка** — токарні верстати, столярні верстаки, фрезери
- **Харчове обладнання** — пивоварні, дистиляційні колони, коптильні, олійні преси
- **Скло та кераміка** — гончарні кола, муфельні печі, тиглі, верстати для різання скла
- **Текстиль** — ткацькі верстати, намотувальні механізми
- **Фільтрація** — мішкові фільтри, циклонні пиловловлювачі, промислова вентиляція
- **Пакування** — пакувальні лінії, дозатори, датувальники
- **Інше** — намотувальні верстати, нестандартне обладнання
